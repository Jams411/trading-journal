## Methodology

- Source: Zerodha Tax P&L statements (taxpnl-EKV211-\*) — one file per FY, all segments combined
- ROI = Net P&L ÷ capital deployed, derived from Zerodha ledger (avg / peak balance per FY)
- Net P&L = Realized P&L − Charges + Other Credits/Debits
- Charges include brokerage, GST, STT, exchange transaction charges, SEBI fees, stamp duty
- Learning-phase ROI (FY2021-22, FY2022-23) not shown — account capital was too small for percentages to be meaningful
- All figures in INR (₹)
