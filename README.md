## Methodology

- **Source data:** Zerodha Tax P&L statements (5 FYs) + Zerodha ledger statements + Angel One Trading Insights + Angel One P&L statements
- **Trade counts:** Row counts from Tax P&L / Trading Insights files — realized trades only, verified
- **Capital denominator:** Monthly avg ledger running balance (Zerodha) · FY avg balance (Angel One)
- **Nifty 50 returns:** Verified via Yahoo Finance (^NSEI) monthly closing prices
- **Zerodha P&L note:** Tax P&L figures are gross (sell value − buy value before charges). Net figures account for brokerage, STT, exchange fees, SEBI fees, stamp duty.
- **Apr 2025:** Excluded from combined series (Zerodha ledger had insufficient data for that month)

*Data as of Aug 2025. Raw statements available on request.*
